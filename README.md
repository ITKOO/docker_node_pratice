# docker_node_pratice

### 1) Docker를 사용한 실행방법

아래 명령어를 입력한 후 실행완료 시 http://localhost:3000/ 에 접속합니다.

```
docker-compose up
``` 

### 2) 로컬에서 node가 설치된 환경에서의 실행방법 

**/config/config.json** 파일에서 **development** 부분을 개인의 DB환경에 맞게 수정 후
```
"development": {
  "username": "", // 계정명
  "password": "", // 비밀번호
  "database": "", // DB명
  "host": "", // 주소 
  "dialect": "mysql"
},
```

아래 명령어를 입력 후 실행완료 시 http://localhost:3000/ 에 접속합니다.

```
node app.js
```
