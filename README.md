# docker_node_pratice


### 📍 실행방법 

**/config/config.json** 파일에서 **development** 부분을 개인의 DB환경에 맞게 수정 후
```
"development": {
  "username": "", // 계정명
  "password": "", // 비밀번호
  "database": "", // DB명
  "host": "", // 주소 
  "dialect": "mysql"
},
```

아래 명령어를 입력하면 실행이 가능합니다.

```
node app.js
```
